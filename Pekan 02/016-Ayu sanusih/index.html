APA ITU HTML?

HTML (HyperText Markup Language) adalah standar bahasa yang digunakan untuk menampilkan teks, gambar, video, dan audio di halaman web. HTML berbentuk file teks yang terdiri dari elemen-elemen bernama tag. Setiap tag HTML ditandai dengan tanda kurang dari (<) dan lebih dari (>), contohnya: <html>, <head>, <body>, <p>, dan lainnya. Tag dalam HTML ada yang berpasangan, ada juga yang tidak. Jika tag memiliki pasangan, maka tag penutupnya diawali dengan garis miring (/), misalnya: </html>, </head>, </body>, </p>, dll.
File HTML bisa dibuat dengan berbagai aplikasi Text Editor dan disimpan dengan ekstensi .html atau .htm. Agar lebih nyaman dalam belajar, disarankan untuk memakai Text Editor yang dirancang khusus untuk menulis kode HTML. Ada banyak Text Editor gratis yang bisa ditemukan di internet untuk keperluan ini.
