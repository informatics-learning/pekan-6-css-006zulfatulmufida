Apa itu HTML?

HTML adalah bahasa markup standar yang digunakan untuk menampilkan teks, gambar, video, dan audio pada halaman web. Dengan HTML, Anda dapat membuat situs web sendiri dengan mudah.

HTML terdiri dari elemen-elemen yang disebut tag. Tag HTML diapit oleh tanda kurung sudut, seperti <html>, <head>, <body>, dan sebagainya. Beberapa tag memiliki pasangan, seperti </html>, </head>, </body>, yang berfungsi sebagai penutup elemen.
Dokumen HTML dapat dibuat menggunakan aplikasi Text Editor apa saja dan disimpan dengan ekstensi .html atau .htm. Untuk pengalaman terbaik, disarankan menggunakan Text Editor yang dirancang khusus untuk menulis kode HTML. Saat ini, banyak Text Editor gratis tersedia di internet yang dapat membantu dalam proses belajar HTML.
