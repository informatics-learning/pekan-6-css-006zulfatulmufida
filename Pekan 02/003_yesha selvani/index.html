APA ITU HTML?
HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat dan mengatur struktur halaman web. HTML menggunakan elemen-elemen yang disebut dengan tag untuk menampilkan teks, gambar, video, dan berbagai elemen lainnya dalam halaman web.
Setiap dokumen HTML terdiri dari berbagai tag seperti <html>, <head>, <body>, dan lain-lain. Beberapa tag memiliki pasangan, seperti <p> dan </p> untuk paragraf, sementara beberapa lainnya tidak memiliki pasangan, seperti <br> untuk baris baru.
