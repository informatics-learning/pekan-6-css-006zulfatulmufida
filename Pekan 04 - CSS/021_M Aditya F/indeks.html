Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat dan menyusun halaman web. HTML bekerja dengan menggunakan elemen-elemen yang membentuk struktur halaman web, seperti teks, gambar, tabel, dan tautan.

Fungsi HTML

  • Menentukan struktur halaman web
  • Menghubungkan halaman web dengan tautan (hyperlink)
  • Mengatur tampilan teks, gambar, tabel, dan elemen lainnya
  • Memudahkan integrasi dengan CSS dan JavaScript

Sejarah HTML

HTML pertama kali dikembangkan oleh Tim Berners-Lee pada tahun 1991 sebagai standar untuk membangun halaman web. Seiring perkembangan teknologi, HTML mengalami banyak pembaruan hingga versi terbaru, yaitu HTML5.

Versi HTML dari Masa ke Masa

Versi Tahun Rilis Fitur Utama
HTML 1.0 1991 Versi pertama, hanya mendukung teks dan link
HTML 2.0 1995 Formulir dan elemen dasar lainnya
HTML 3.2 1997 Dukungan untuk tabel dan skrip
HTML 4.01 1999 Pemisahan antara struktur (HTML) dan tampilan (CSS)
HTML5 2014 Tag video, audio, canvas, dan API baru
