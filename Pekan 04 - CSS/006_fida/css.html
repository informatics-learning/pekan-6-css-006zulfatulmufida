APA ITU HTML?

HTML (HyperTextMarkupLanguage) adalah bahasa standar yang digunakan untuk menampilkan teks, gambar, video, dan audio di halaman web. HTML terdiri dari elemen-elemen yang disebut dengan tag. Setiap tag HTML diawali dengan tanda kurang dari (<) dan lebih dari (>), misalnya: <html>, <head>, <body>, dan <p>.

Dokumen HTML dapat dibuat menggunakan aplikasi Text Editor apa saja dan disimpan dengan ekstensi .html atau .htm. Untuk mempermudah belajar, disarankan menggunakan Text Editor khusus untuk HTML yang tersedia gratis di Internet.
