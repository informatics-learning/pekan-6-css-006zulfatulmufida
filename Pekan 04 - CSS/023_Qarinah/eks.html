Cascading Style Sheet

Cascading Style Sheet(css) adalah sebuah pemrogaman atau boleh dibilang script yang mengendalikan beberapa komponen (taghtml) dalam sebuah website sehingga tampilan akanmenjadi lebih terstruktur dan seragam

CSS) merupakan aturanuntuk mengendalikan beberapa komponen dalamsebuah web sehingga akan lebih terstruktur dan seragam
