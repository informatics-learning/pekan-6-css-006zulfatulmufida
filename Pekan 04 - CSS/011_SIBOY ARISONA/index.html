APA ITU HTML?

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk menampilkan teks, gambar, video, dan audio di dalam halaman web. HTML merupakan file teks yang tersusun atas elemen-elemen yang disebut dengan tag. Tag HTML diawali dengan tanda lebih kecil (<) dan tanda lebih besar (>), misalnya: <html>, <head>, <body>, dan <p>.
Tag HTML ada yang memiliki pasangan dan ada juga yang tidak. Jika suatu tag memiliki pasangan, maka tag penutup akan disertai dengan tanda slash (/), misalnya: </html>, </head>, </body>, dan </p>.
Dokumen atau file HTML dapat dibuat menggunakan aplikasi Text Editor apa saja dan disimpan dengan ekstensi .html atau .htm. Untuk memudahkan proses belajar, disarankan menggunakan Text Editor yang dirancang khusus untuk menulis kode HTML. Saat ini, banyak software Text Editor untuk HTML yang tersedia secara gratis di internet.
